Атрибуты можно обрабатывать аналогично элементам.

Преобразование 1

Исходный XML
<source>

<employee id="js0034"> Joe Smith </employee>

</source>

Результат
Joe Smith
[<b>
  <i>js0034</i>
</b>]

Представление HTML
Joe Smith [ js0034 ]
Преобразование XSLT
<xsl:stylesheet version = '1.0'
     xmlns:xsl='http://www.w3.org/1999/XSL/Transform'>

<xsl:template match="employee">
     <xsl:value-of select="."/>
     <xsl:text>[</xsl:text>
     <xsl:apply-templates select="@id"/>
     <xsl:text>]</xsl:text>
</xsl:template>

<xsl:template match="@id">
     <b>
          <i>
               <xsl:value-of select="."/>
          </i>
     </b>
</xsl:template>


</xsl:stylesheet>

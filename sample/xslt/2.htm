Каждое преобразование XSL должно начинаться с элемента xsl:stylesheet. Атрибут version='1.0' определяет версию спецификации XSL. Этот пример представляет собой простейшее возможное преобразование. Поскольку оно не содержит чего-либо помимо заголовка, то используется обработка по умолчанию.

Преобразование 1

Исходный XML
<source>

<em>Hello, world</em>

</source>

Результат
Hello, world

Представление HTML
Hello, world
Преобразование XSLT
<xsl:stylesheet version = '1.0'
     xmlns:xsl='http://www.w3.org/1999/XSL/Transform'>


</xsl:stylesheet>

Метод вывода "text" выводит результирующее дерево посредством вывода строковых значений каждого текстового узла в порядке следования в документе без каких-либо изменений.

Преобразование 1

Исходный XML
<source>

<AAA id="12"/>

</source>

Результат
<!ELEMENT AAA ANY><!ATTLIST AAA id ID #REQUIRED>Look at my source in your browser

Преобразование XSLT
<xsl:stylesheet version = '1.0'
     xmlns:xsl='http://www.w3.org/1999/XSL/Transform'>

<xsl:output method="text"/>
<xsl:template match="AAA">
     <xsl:text><!ELEMENT </xsl:text>
     <xsl:value-of select="name()"/>
     <xsl:text> ANY></xsl:text>
     <xsl:text><!ATTLIST </xsl:text>
     <xsl:value-of select="name()"/>
     <xsl:text> </xsl:text>
     <xsl:value-of select="name(@*)"/>
     <xsl:text> ID #REQUIRED></xsl:text>
     <xsl:text>Look at my source in your browser</xsl:text>
</xsl:template>


</xsl:stylesheet>
